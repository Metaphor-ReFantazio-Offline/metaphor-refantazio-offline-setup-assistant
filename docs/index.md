## ğŸ› ï¸ Troubleshooting {#troubleshooting}

### ğŸš¨ Common Issues

#### Installation Problems

**Issue: "Cannot find game installation"**
- **Solution**: Navigate to your Metaphor: ReFantazio folder manually
- **Common Locations**:
  - Steam: `C:\Program Files (x86)\Steam\steamapps\common\Metaphor ReFantazio`
  - Epic: `C:\Program Files\Epic Games\Metaphor ReFantazio`

**Issue: "Missing dependencies"**  
- **Solution**: Install Microsoft Visual C++ Redistributables
- **Download**: [Microsoft Official Site](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist)

#### Performance Issues

**Issue: Low FPS despite good hardware**
- **Check**: Background applications consuming resources
- **Solution**: Close unnecessary applications, lower graphics preset
- **Advanced**: Adjust render scale to 80-90%

**Issue: Audio crackling or delays**
- **Check**: Audio device sample rate conflicts
- **Solution**: Set all audio devices to same sample rate (48kHz)
- **Windows**: Disable audio enhancements in Sound Control Panel

#### Controller Issues

**Issue: Controller not detected**
- **Windows**: Install Xbox accessories app or DS4Windows
- **macOS**: Use built-in controller support or controlly
- **Linux**: Install xpad kernel module or configure jstest

**Issue: Button mapping not working**
- **Solution**: Reset controller configuration to defaults
- **Advanced**: Manually edit controller config file
