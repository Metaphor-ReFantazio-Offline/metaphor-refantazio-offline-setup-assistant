## 🎮 How to Use

### Quick Start
1. **Download** the setup assistant from the link above
2. **Extract** using 7-Zip, WinRAR, or built-in extractor
3. **Launch** the assistant interface
4. **Point** to your Metaphor: ReFantazio installation directory
5. **Configure** your preferred settings
6. **Click Launch** to start offline mode

### Configuration Options

| Setting | Options | Description |
|---------|---------|-------------|
| Display | 1080p, 1440p, 4K | Resolution and aspect ratio |
| Graphics | Low, Medium, High, Ultra | Performance vs quality presets |
| Controls | Keyboard, Controller, Hybrid | Input method configuration |
| Audio | Stereo, 5.1, 7.1, Headphones | Sound output optimization |
| Mode | Story, Exploration, Training | Different offline experiences |
