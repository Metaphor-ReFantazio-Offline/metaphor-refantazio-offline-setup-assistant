## 🎯 How to Use

### 🚀 First Time Setup
1. **Launch** the Metaphor Setup Assistant
2. **Browse** and select your game installation folder
3. **Choose** your display resolution and refresh rate

5. **Pick** a performance profile
6. **Click "Launch Game"** and enjoy!
